Security

Prompt Injection

Monitor has no API keys. Injection from Reddit content is harmless.

Content Exfiltration

Publisher has no Claude. Cannot send article content to an LLM endpoint.

Runaway Publishing

Nothing auto-publishes. User must approve via email. Writer cannot reach Framer.

Credential Exposure

GCP Secret Manager for prod. Env vars injected at runtime, never on disk.

Repeated Bad Content

Rejection memory across sessions. Agent learns from past feedback.

Supply Chain

PyPI access via policy. Package requests escalated to email for approval.
